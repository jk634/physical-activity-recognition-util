ActivityDao

class ActivityDao(dbHelper: TrainingDbHelper)

ActivityDao provides methods to interact with the database to manage activities and training data. It encapsulates database operations related to activities and training data such as saving activities, retrieving activity lists, fetching training data for activities, and deleting activities and their associated samples. This class acts as a Data Access Object (DAO) for managing activities and training data.
