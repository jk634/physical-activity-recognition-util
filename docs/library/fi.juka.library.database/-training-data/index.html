TrainingData

class TrainingData(val id: Long, val x_axis: Float, val y_axis: Float, val z_axis: Float, val total_acceleration: Float, val timestamp: Long, val activityId: Long)

TrainingData represents a single data entry for training purposes, including acceleration values in x, y, and z axes, total acceleration, timestamp of the data, and the associated activity ID.
